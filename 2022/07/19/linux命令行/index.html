man的使用

man命令用来访问手册，q键用于quit

不记得命令名的话可以用关键字搜索手册页，man -k key,
查找与终端相关的命令就可以 man -k terminal

linux手册页的内容区域

区域号 所涵盖的内容
1 可执行程序或shell命令
2 系统调用
3 库调用
4 特殊文件
5 文件格式与约定
6 游戏
7 概览、约定及杂项
8 超级用户和系统管理员命令
9 内核例程

man xterm 的话其实是 man 1 xterm

man 3 intro就是阅读intro的第三部分

浏览文件系统

cd /usr/bin 指向usr目录包含的bin目录下的用户二进制文件

.. 返回上级目录

cd 直接跳回主目录

pwd 查看当前目录

文件和目录列表

ls 显示当前目录下的文件和目录

注意ls按列排序

ls -F会在目录名后面加上正斜线，在可执行文件后面加上星号

ls -a会把隐藏文件也显示出来，而且前面会有.号

ls -F -R 递归式列出包括了子文件夹下面的文件

ls -FR和ls -F -R是一样的效果

ls -l 长列表效果，有更多信息

指定特定文件的名称作为过滤器时

？匹配一个字符

  • 匹配零个或者多个字符

ls -l f[a-i]ll 就是说文件名的第二个字符从a到i都是可以的

ls -l f[ai]ll 意思是文件名的第二个字符只能是a或者i

ls -l f[!a]ll 表示不能是a

处理文件

查看文件信息

file /bin/ls

可以用来查看ls文件是不是ELF格式，也就是 Executable and Linking Format

显示文件内容

cat命令

可以显示一个或多个文件的内容

cat file1 file2 会将file1和file2的内容拼接起来显示然后退出

创建文件

touch test_one 在当前目录下创建一个test_one

再来一次touch test_one 可以改变文件的修改时间

ls -l 默认显示的是修改时间，如果想显示访问时间的话

ls -l —time=atime test_one 就可以显示最后一次访问的时间了

复制文件

语法： cp source destination

cp file1…fileN dir

可以把file1到fileN的文件全部复制到dir目录

cp -i test_one test_two

会强制询问是否覆盖已有的文件，不然就是如果文件已经存在，默认覆盖

cp -R 可以递归地复制整个文件夹中的内容

cp -R scripts/ Mod_scripts可以把scripts文件夹下面的所有内容都复制到Mod_scripts中

重命名文件

mv命令

mv file1 file2

把文件file1重命名为file2

mv file1 … fileN dir 把多个文件移动到某个目录

链接文件

ln -s data_file sl_data_file 就创建了一个符号文件sl_data_file,它指向data_file,但这两个文件中的内容各不相同，两个文件的inode编号也不一样

ls -i *data_file 就可以查看他俩的inode编号了

tab键可以自动补全

删除文件

rm file

处理目录

设置当前工作目录

cd dir

创建新目录

mkdir dir

删除目录

rmdir dir

注意如果目录非空，那么上面的命令会失效，rm -rf可以删除一个目录以及其中的所有内容

中间命令

grep命令

显示文件（或输入流）中与参数匹配的行

grep root /etc/passwd 会显示文件passwd中包含文本root的所有行

grep root /etc/*

可以查看/etc目录中所有包含root的文件

grep -i 不区分大小写

grep -v 反转匹配，即显示所有不匹配的行

less命令

内容分屏显示

grep ie /usr/share/dict/words | less

less 经常和别的命令搭配起来一起使用

diff命令

用来查看两个文件的不同

find命令

find dir -name file -print

可以再目录中寻找文件

locate与find相似，但是locate可以再系统创建的文件索引中查找文件

find与tail命令

head命令默认显示文件的前10行内容，tail则是后10行

可以指定参数 head -5 就是前五行，tail +5 就是后五行

sort命令

将文件内的所有行按照字母顺序快速排序， -n 可以按照数字顺序

  • r可以进行反向排序
